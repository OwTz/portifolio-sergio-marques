# Portfólio Sérgio Marques

Bem-vindo ao meu portfólio!

## Sobre o Projeto
Este projeto foi desenvolvido para apresentar meu trabalho como desenvolvedor Full Stack, especialista em Cyber Security e soluções digitais. Aqui você encontra minhas principais habilidades técnicas, serviços oferecidos e formas de contato profissional.

O site é moderno, responsivo e utiliza tecnologias atuais como React, Vite, TailwindCSS e animações com Framer Motion. O objetivo é divulgar meu trabalho, facilitar o contato com clientes e mostrar minha experiência em desenvolvimento e segurança.

## Principais Seções
- **Início:** Apresentação pessoal e chamada para ação.
- **Skills:** Linha do tempo interativa com minhas principais habilidades técnicas.
- **Serviços:** Soluções que ofereço, como desenvolvimento web, segurança e design.
- **Contato:** Links para redes sociais e WhatsApp.

## Tecnologias Utilizadas
- React
- Vite
- TailwindCSS
- Framer Motion
- Lucide Icons

## Como rodar o projeto
```bash
pnpm install
pnpm run dev
```
